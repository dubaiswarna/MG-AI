üìù STEP 6: CREATE SERVER.JS FILE
=============================================

In cPanel File Manager:

1. Navigate to /home/goldengod/public_html/

2. Click "New File"

3. Name: server.js

4. Right-click server.js ‚Üí "Edit"

5. Paste this code:

```javascript
const { createServer } = require('http')
const { parse } = require('url')
const next = require('next')

const dev = false
const hostname = 'localhost'
const port = process.env.PORT || 3000

const app = next({ dev, hostname, port })
const handle = app.getRequestHandler()

app.prepare().then(() => {
  createServer(async (req, res) => {
    try {
      const parsedUrl = parse(req.url, true)
      await handle(req, res, parsedUrl)
    } catch (err) {
      console.error('Error occurred handling', req.url, err)
      res.statusCode = 500
      res.end('internal server error')
    }
  }).listen(port, (err) => {
    if (err) throw err
    console.log(`> Ready on http://${hostname}:${port}`)
  })
})
```

6. Click "Save Changes"

7. ‚úÖ server.js created!

‚ö†Ô∏è TROUBLESHOOTING
=============================================

PROBLEM 1: Application Won't Start
-----------------------------------
Solution:
- Check Node.js version (use 18.x or 20.x)
- Check all files uploaded correctly
- Check server.js exists
- Check environment variables are set
- Look at application logs in cPanel

PROBLEM 2: Database Connection Error
-------------------------------------
Solution:
- Verify DATABASE_URL is correct
- Check username/password
- Verify database exists
- Check remote MySQL is enabled
- Test connection from cPanel phpMyAdmin

PROBLEM 3: "Cannot find module 'next'"
---------------------------------------
Solution:
- Run npm install again
- Make sure node_modules uploaded or installed
- Check package.json exists

PROBLEM 4: 502 Bad Gateway
---------------------------
Solution:
- Application may be crashed
- Check error logs
- Restart application
- Verify Node.js version

PROBLEM 5: Blank Page
----------------------
Solution:
- Check .next folder uploaded
- Check public folder uploaded
- Check browser console for errors
- Verify environment variables
